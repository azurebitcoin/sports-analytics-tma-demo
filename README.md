# SportSignal TMA Demo

Client-facing demo for the Telegram Mini App sports analytics platform.

## What Is Included

- React + Vite frontend only
- Mocked sports fixtures, predictive cards, and access tiers
- Browser-preview mode outside Telegram
- No backend code
- No infrastructure or production integration setup

## What Is Not Included

- FastAPI backend
- PostgreSQL / Redis setup
- Real sports data provider adapters
- Real paywall verification
- Real Telegram Bot API delivery
- LLM provider credentials and production prompts

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

This repository is intentionally limited to a presentation-safe demo surface.
